# Sanity SvelteKit demo

## Sanity

1.  `cd studio`
2.  `npm i`
3.  Run `sanity dev`

## SvelteKit

1.  `cd web`
2.  `npm i`
3.  Run `npm run dev -- --open`

## Notes

- CORS Origins issue. Add `http://localhost:3000` in your Sanity dashboard > API settings (allow credentials)
- Requires node 12.5+ (node 14 recommended)
